# SunCart Store

A complete E-commerce web application built for summer essentials.

## Features
- Dynamic Navbar with Authentication (Login/Logout)
- Protected Product Details Page
- Add to Cart functionality with Login Check
- User Profile Management (Update Info, Phone, Photo)
- Responsive Design for Mobile and Desktop

## Technologies Used
- Next.js
- Tailwind CSS
- Context API (for Auth and Cart)
- JSON Data for Products

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
